Experiment 6: Curve Fitting and Regression Analysis
Overview:

This experiment focuses on representing real-world data using curve fitting techniques. The objective is to determine a mathematical function that best fits a given dataset and to understand how different models affect accuracy.

Methodology:
Given dataset of input-output values analyzed
Manual implementation of least squares method:
Computation of required summations
Determination of slope and intercept for best-fit line
MATLAB implementation:
Linear curve fitting
Polynomial (second-degree) fitting

Visualization:
Plotting raw data along with fitted curves
Comparison between manual and computational results

Results:
Linear model provided a basic approximation of the data
Polynomial model showed better accuracy and closer fit
MATLAB results were consistent with manual calculations
Observed that higher-order models can capture data trends more effectively
Gained understanding of model selection and error minimization
